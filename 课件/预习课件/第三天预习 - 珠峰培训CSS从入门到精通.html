1.webstorm的快捷键

  • File(文件) - open(打开)
  • New window 表示在新的窗口打开项目文件
  • File - Settings - font 可以修改编辑器的字体大小
  • 如果字号是禁灰的状态 就把save as(另存为)点击一下就可以
  • F12 或者 鼠标右键 - 检查 可以调出控制面板
  • code - Reformat code 代码格式化 ctrl + shift + /
  • 注释的快捷键(注释的内容只是为了方便开发者去阅读 不会在浏览器中解析)
  • ctrl + / 单行注释
  • ctrl + shift + / 多行注释
  • ctrl + C 复制
  • ctrl + V 粘贴
  • ctrl + D 复制整行代码
  • ctrl + Z 返回上一步 撤销
  • 双击 选中所有内容
  • shift + enter 光标快速跳到下一行
  • ctrl + alt + L 代码格式化
  • shift + F6 文件重命名
  • tab -> 标签名 / 样式属性

3.CSS样式属性

1) 默认样式自带样式属性的标签
  • a 标签有默认自带的下划线
  • ul li / ol li 前面有默认的小黑点 / 数字
  • h1-h6 默认自带加粗
  • strong/ b 自带加粗效果
  • i / em 默认自带斜体
  • 浏览器有默认自带的间距(谷歌浏览器body默认的间距是 8px)
2) 去掉默认样式自带样式属性
  • text-decoration: none; 去掉a默认的下划线
  • text-decoration: underline; 添加下划线
  • list-style: none; 去掉列表(ul ol /li)默认的样式
  • font-weight:normal; 把粗体变成细体
  • font-style: normal; 把斜体变成正常字体
  • font-weight: bold; 把默认字体加粗
3) 文字的样式添加技巧
  • 字体大小 font-size
  • 字体颜色 color
  • 字体粗细 font-weight
  • 行高和间距 line-height/margin/padding
  • 高度(height)和行高(line-height)的值一样，文字内容垂直居中;
4) margin 和 padding的区别

1.是否对元素在浏览器中的大小产生影响：

  • margin：外边距是不会影响元素显示在浏览器中的实际大小，只是产生了间距;
  • padding: 内边距会影响元素实际显示在浏览器中的实际大小padding是算在元素的实际大小里的

2.含义和实际应用：

  • margin : 外补丁（外边距） - 跟另外一个盒子的间距(大部分是并列的两个盒子)
  • padding: 内补丁（内边距）- 父元素和子元素的间距（大部分是嵌套的两个盒子）

4.元素居中的技巧

1.文字居中

  • 文字水平居中: text-align: center；
    • text-align: left; 文字居左显示
    • text-align:center; 文字居中显示
    • text-align:right; 文字居右显示
  • 文字垂直居中：height高度和line-height行高的值一样，文字就垂直居中
    • 用来控制一行文字和另一行文字之间的垂直间距；值越大，间距越大；
    • 当行高跟高度的值一样的情况下，文字就垂直居中了；

2.图片/ 块级元素居中

  • 水平居中: width宽度 / margin: 0 auto(上下 左右);
  • 垂直居中: 定位
    3.letter-spacing 用来控制文字与文字之间的水平(左右)间距，值越大小，间距越大

5.出现省略号必备的条件（四个）

  • width: 200px; 宽度
  • white-space: nowrap; 禁止文字内容折行
  • overflow: hidden; 溢出隐藏:把超出大小的内容截掉
  • text-overflow: ellipsis; 让文字内容以省略号的方式呈现
