5.网页的基本组成部分

1、结构层 HTML / HTML5
网页的骨架，HTML是网页内容的载体，内容就是网页制作者放在页面上想要让用户浏览的信息，可以包含文字、图片、视频等。
2、样式层 CSS/ CSS3
CSS样式是表现。就像网页的外衣，比如，标题字体、颜色变化，或为标题加入背景图片、边框等，所有这些用来改变内容外观的东西称之为表现。
3、行为层 Javascript
JavaScript是用来实现网页上的特效效果。如：鼠标滑过弹出下拉菜单。或鼠标滑过表格的背景颜色改变。还有焦点新闻（新闻图片）的轮换。可以这么理解，有动画的，有交互的一般都是用JavaScript来实现的。

6.HTML基础

1、网页的基本组成

  • html（Hypertext Markup Language） - 是网页的骨架、结构
  • css（Cascading Style Sheets）层叠样式表 - 样式：用来修饰结构的
  • js（javascript）- 行为/动态交互

2、HTML(超本文标记语言)
网页文件本身是一种文本文件（除了文字再无其他），只有通过特定的标记描述，那些普通文本才能在浏览器中正常的显示，在页面中如果遇到图片，音视频的时候如何用文本描述，需要用超文本进行引入才可以正常显示;

3、HTML文件的特点
浏览器根据标记来解析和描述内容（从html的第一行开始向下解析），如果遇到错误，浏览器不会指出，也不会报错，会跳过此处继续加载，编辑人员需要根据显示效果去分析哪里出错误了 – html不会报错

4、html的兼容问题
因为浏览器是由不同厂商生产的，所以对html的解析有分歧，产生了页面效果显示不一致的情况，我们将这个问题，叫做兼容问题。

5、标签语法
由成对的尖括号和关键字(标签名 tagName)组成，叫做标签。
关键字(标签名)：是已经规范过的，不是自己随便制造的。

6、标签属性
属性提供了有关HTML元素的更多的信息
语法：由属性名和属性值组成(键值对 key=’value’)
一个标签可以有很多标签属性，每一个属性之间用空格隔开

<span style="color:red;" title="标题"></span>



span  表示的是标签名

style 表示的是标签属性名

"color:red;" 表示的标签属性值

7、标签的分类
开始标签 <div>
结束标签 </div>
自闭合标签(空标签) <br/>

8、HTML的元素
可以将元素理解成一个容器，容器中放置的就是我们网页中的主要内容, 内容可以为空;
由标签和内容组成的叫做元素;
<div>此处是内容或者内容为空</div>

9、元素特点
由开始标签和结束标签组成的元素，可以进行嵌套(一个元素包着另一个元素)，元素中可以添加内容。
空标签，只能进行功能的添加，例如img元素，用来引入图片，br用来进行换行
<div></div> <br/>

10、元素分类
标题类 <h1></h1>.....<h6></h6>
文字段落 <p></p>
链接跳转(超链接) <a></a>

  • href=’要跳转的链接地址’
  • 禁止a标签的链接跳转的行为 javascript:;
  • target=”_blank” 在新窗口打开，当前的窗口还在
  • target=”_self” 在当前窗口打开，当前窗口被覆盖

加粗：<strong></strong><b></b>
斜体: <em></em> <i></i>
列表类

  • 无序列表 (前面有一个小黑点) <ul><li></li></ul>
  • 有序列表 (前面是数字) <ol><li></li></ol>
  • 定义列表 <dl><dt></dt><dd></dd></dl>
    • dl 定义列表
    • dt 定义列表的标题内容
    • dd 定义列表描述内容
    • dl>dt>dd(dd可以出现很多个)

div 用来划分大块结构 <div></div>
划分小块结构和文字 无意义 <span></span>
插入图片 <img />

  • src=”” 放图片的路径地址
  • alt=”” 放图片的描述文字内容 (图片路径错误或者不存在)

br 表示折行 单标签 它属于自闭合标签 <br/>
sub 表示下标 <sub></sub>
sup 表示上标 <sup></sup>
input 表单元素 <input />

  • input 里面有一个type的属性 表单的类型
  • type=”text” 文本类型
  • type=”password” 密码类型
  • type=”button” 按钮类型
  • type=”radio” 单选按钮类型
  • type=”checkbox” 多选按钮类型
  • type=”reset” 多选重置类型
  • type=”submit” 提交按钮类型
