1. 简述什么叫做标签语义化

含义：标签语义化：合理的标签做合理的事情。
好处 :

  • 1、利于SEO优化（也就是搜索引擎的抓取，搜索引擎的爬虫也依赖于标记来确定上下文和各个关键字的权重）；
  • 2、在样式丢失的时候，还是可以比较好的呈现结构；
  • 3、更好的支持各种终端，例如无障碍阅读和有声小说等；
  • 4、利于团队开发和维护，W3C给我们定了一个标准，那么团队中都遵循这个标准，那么代码的差异就会缩小，在开发和维护的时候就可以提高效率；

日常工作中怎样遵循标签语义化：

  • 1、尽量减少使用无意义标签，例如span和div；
  • 2、尽量不使用标签本身的css属性，例如b、font、s 等标签，如果需要这些样式，那么使用css样式来进行添加；
  • 3、在需要强调的部分，使用strong、em，但是样式尽量使用css样式来描述；
  • 4、表格搭建时，使用<thead>表格头部</thead> <tbody>表格身体</tbody>表格尾部`；
  • 5、列表搭建时，使用<ul>无序列表</ul> ``<ol>有序列表</ol> ``<dl>定义列表</dl>；

2.HTML中块级标签和行内标签的特点

块级元素 div ul ol li dl dt dd h1-h6 p...

  • 天生自带属性display: block
  • 独占一行；
  • 排列方式: 从上到下垂直排列
  • 可以设置宽高以及盒子模型的其他属性(width/height/margin/padding/border)属性
  • 如果块级元素不设置宽高，宽度会继承他父级元素的宽度，高度是本身内容的高度;

行内元素 a b strong img span em i...

  • 天生自带属性display: inline
  • 不独占一行;
  • 排列方式: 从左到右依次排列，超出父级元素的大小会自动折行
  • 设置宽度属性不起作用，宽高是本身内容的大小；要宽度起作用就需要转化为具有块级元素的特征 display:block
  • 设置内外边距的左右起作用，但是上下不起作用 display:block；
  • 如果行内元素不设置宽高，高是本身内容的大小;

3.使用css,让一个div消失在视野中；发挥想象力

  • display:none 让元素隐藏起来
  • visibility: hidden; 让所有可见性的元素隐藏
  • width:0 + overflow:hidden
  • height:0 + overflow: hidden
  • position: absolute /relative /fixed + 方位top/bottom/left/right: - 9999px;
  • margin-top/bottom/left/right: -9999px;
  • transform: translateX(-9999px) 、translateY(-9999px);
  • opacity: 0; 透明度 【0-1】
  • transform: scale(0); 缩放比为0
  • background-color / color : transparent;

4.用css实现三栏布局；左定右定中间自适应(双飞翼，圣杯 )

参考网站: https://m.jb51.net/css/456931.html

1.利用定位position实现

<style type="text/css">

.main{position:relative;padding:0 210px;}     .left{position:absolute;left:0;top:0;width:200px;height:500px;background-color:red}       .right{position:absolute;right:0;top:0;width:200px;height:500px;background-color:red}

.middle{height:500px;background-color:#000}

</style>

<div class="main">

    <div class="left"></div>

    <div class="middle"></div>

    <div class="right"></div>

</div>

2.利用浮动的原理去实现

//=> float + margin

<style> 

.main{padding:0 210px;}   .left{float:left;width:200px;height:500px;background-color:red}       .right{float:right;width:200px;height:500px;background-color:red}

.middle{height:500px;margin:0 210px;background-color:#000}

</style>

<div class="main">

    <div class="left"></div>

    <div class="right"></div>

    <div class="middle"></div>

</div>

//=> float + margin + calc

<style>

.left{float:left;width:200px;height:500px;background-color:red}

.middle{float:left;width: calc(100% - 420px);height:500px;background-color:#000;margin:0 10px;}    

.right{float:left;width:200px;height:500px;background-color:red}

</style>

<div class="main">

    <div class="left"></div>

    <div class="middle"></div>

    <div class="right"></div>

</div>

//=> float +  margin负值（width:100%）

<style> 

   .left{position:relative;float:left;width:200px;height:500px;background-color:red}

.middle{float:left;width:100%;margin:0 -200px;height:500px;background-color:#c8dcf2;padding:0 200px;box-sizing: border-box}    .right{float:left;width:200px;height:500px;background-color:red}

</style>

<div class="main">

    <div class="left"></div>

    <div class="middle"></div>

    <div class="right"></div>

</div>

//=> display：inline-block +  margin + calc

.main{font-size:0;} /* 去掉行内块受换行符的影响*/

.left{display:inline-block;width:200px;height:500px;background-color:red}

.middle{display:inline-block;width: calc(100% - 420px);height:500px;background-color:#000;margin:0 10px;}

.right{display:inline-block;width:200px;height:500px;background-color:red}

</style>

<div class="main">

    <div class="left"></div>
